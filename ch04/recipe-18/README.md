# Recipe 18: Setting Up Playwright MCP

You want Claude Code to interact with web browsers -- navigate pages, click elements, fill forms -- but you haven't configured Playwright MCP yet.

## Files

- `.mcp.json` — Basic Playwright MCP configuration (headed mode, default browser)
- `mcp-headless.json` — Headless variant with `--browser chrome` for CI/CD and server environments
