# Claude Code Cookbook

**100 Practical Recipes for MCP, Skills, Hooks & Sub-Agents**

Available on [Amazon Kindle](https://www.amazon.com/dp/B0GM9QFGXJ).

## About This Repository

This repository contains companion sample code for the book *Claude Code Cookbook*. Each recipe directory maps directly to a recipe in the book.

## How to Use

1. Find the recipe number you're reading in the book
2. Navigate to the corresponding `chXX/recipe-XX/` directory
3. Copy the configuration files into your own project
4. Adapt paths, project names, and details to your environment

Most files are configuration (JSON, YAML, Markdown) that you drop into your project. Code files (TypeScript, Python, shell scripts) are complete and ready to run.
