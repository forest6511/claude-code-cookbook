  Investigate module structure and
  dependencies. Use proactively for
  codebase exploration.
tools: Read, Grep, Glob
model: haiku
---

Investigate the specified module and report:

1. Directory structure and file listing
2. Entry points (export/import relationships)
3. External library dependencies
4. Interfaces with other modules
5. Test coverage status

Format: Use headings for each item.
Reference code as `file:line`.
