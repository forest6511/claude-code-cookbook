# Recipe 77: Specialist Role Pattern (PM / Architect / Developer / QA)

Having one agent handle requirements, design, implementation, and testing leads to inconsistent quality across phases.

## Files

- `.claude/agents/pm-agent.md` -- Product manager with persistent memory for specs and user stories
- `.claude/agents/architect-agent.md` -- Software architect with persistent memory for design decisions
- `.claude/agents/developer-agent.md` -- Developer with acceptEdits permission for implementation
- `.claude/agents/qa-agent.md` -- QA engineer for testing and quality verification
